The issue

We would like to start plotting the various components of the dataframe and while we can do this without any issues using ggplot there are a few annoyances:

  • The graphs are not interactive. I cannot move my mouse over a data point and see it's value.

  • The process is time consuming. Every time I change something, I have to rerun the code and look at a static plot.

  • Especially cumbersome when datasets are extremely large

The pitch

Let's utilize the power of Shiny. This will increase efficiency.

What will it do?

  • Plot variables

  • Color by a a variable

  • Add a simple regression line

  • Color the regression line

  • Implements mouse interactivity (zooming, etc...)

The shiny app can be found here: http://kwr-3.shinyapps.io/mtcars/

To use the app simple click on your desired preferences on the left hand side.
